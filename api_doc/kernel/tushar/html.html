public static bool getPath(\kernel\requesr request, string file, string type)

Loads files for css/js in following priority order. 1. Load resource file from /webroot/skins/$request->skin/$request->theme/ 2. If Not Found Load file from /webroot/module/$request->module/ 3. If Not Fiund Load file from /webroot

Returns returns false on failure else return valid path

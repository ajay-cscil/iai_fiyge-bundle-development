Class

kernel\behaviour\tree

public class tree extends \kernel\behaviour

This class implements following algorithms and dependent logic used by models 1. pre-order tree traversal [has overhead of updating on change of position] 2. materialized tree traversal with order [has overhead of updating names of all children on change of parent name] 3. simple tree traversal [less write overhead, but higher (at times recursive) overhead on retrive]

Author Tushar Takkar
